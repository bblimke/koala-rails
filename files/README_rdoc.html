koala-rails

Installer and helpers for using Koala with Rails 3

Installation

In your Gemfile 

  gem "koala-rails"

From the command line 

  bundle install
  rails g koala:install

Edit config/facebook.yml to reflect your Facebook application’s ID and secret.
